# sharepoint-list-plugin

获得 OneDrive 文件下载直链的 TamperMonkey 插件。推荐使用 Tampermonkey Beta。

## 构建

```shell
npm run build
```

安装 `dist/sharepoint-list-plugin.user.js` 文件。
